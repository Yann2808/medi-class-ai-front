## ✨ Fonctionnalités
- **Saisie intuitive** de comptes-rendus médicaux
- **Visualisation des résultats** par spécialité
- Historique des analyses
- Mode sombre/clair

## 🏗 Installation Locale
```
git clone https://github.com/votre-user/medi-class-ai-front.git
cd medi-class-ai-front
npm install
npm run dev
```
